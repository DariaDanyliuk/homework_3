HTML

The HyperText Markup Language, or HTML -

- is the standard markup language for documents designed to be displayed in a web browser.

Inline tag names:

  • span
  • button
  • img
  • a

Block tag names:

  • h1
  • p
  • ul
  • body
  • div
  • footer
  • header
  • head
  • html
  • ol

  1. head

    The "head" element is a container for metadata (data about data) and is placed between the "html" tag and the "body" tag. HTML metadata is data about the HTML document. Metadata is not displayed. Metadata typically define the document title, character set, styles, scripts, and other meta information.

  2. title

    The "title" element defines the title of the document. The title must be text-only, and it is shown in the browser's title bar or in the page's tab. The "title" element is required in HTML documents! The contents of a page title is very important for search engine optimization (SEO)! The page title is used by search engine algorithms to decide the order when listing pages in search results.

  3. body

    The "body" HTML element represents the content of an HTML document. There can be only one "body" element in a document.
